Anime Search

If you are unfamiliar with anime, try typing in some of my favorites into the search bar!


Soul Eater


Fire Force


Frieren


Call of the night


Lycoris Recoil


I used the ${encodeURIComponent()} to allow user input. The user inputs a SFW Anime title, and then title is trimmed to remove spaces and inserted into the API url to retrieve the appropriate data (image, title, and description). Users are also able to search shows using both the english and japanese names. This took me forever to figure out but it is so cool!!! - shout out to MDN
